# Online Analysis Track (Live Systems)

This directory contains lab descriptions for live host analysis in a cyber range that simulates an enterprise network.

Labs focus on:

- Rapid triage of suspicious hosts  
- Identification of persistence and C2  
- Detection of credential theft and lateral movement  
- Multi-host incident reconstruction  

---

## Labs Included

- `lab-basic-foothold.md`  
- `lab-privesc-cred-dump-lateral.md`  
- `lab-powershell-intrusion.md`  
- `lab-data-exfiltration.md`  
- `lab-multihost-investigation.md`  

Each lab describes:

- Scenario context  
- Injected artifacts or behaviors  
- Required tools and commands  
- Tasks and deliverables  
- Optional scoring using `../rubrics/scenario-rubric.md`
